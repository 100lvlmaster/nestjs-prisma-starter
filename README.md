# Nest Prisma Starter

## Nest
Run Nest Server in Development mode:

```bash
npm run start

# watch mode
npm run start:dev
```

Run Nest Server in Production mode:

```bash
npm run start:prod
```

## Graphql

Generate typings for the Nest Server:

```bash
npm run typings
```

Typings are generated to `src/generated/graphql.ts`.
